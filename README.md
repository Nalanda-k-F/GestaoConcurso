Desenvolver um aplicativo Blazor que permita a inscrição de usuários e o lançamento das notas obtidas por eles em cursos. O projeto deve ser implementado com base no modelo relacional lógico do banco de dados fornecido.

O sistema deverá conter as seguintes funcionalidades:

Página para inscrever o usuário (o operador realizará a inscrição do candidato).
Página para cadastrar um concurso.
Página para cadastrar as disciplinas.
Página para lançar as notas dos candidatos.
Página para consultar em quantos concursos o usuário está inscrito. (ter a opção para imprimir lista de inscritos)
Página para consultar as notas de um candidato.
Página para imprimir a lista de aprovados, classificada pela média das notas das disciplinas cadastradas para o concurso.
Grupo de até 4 pessoas
